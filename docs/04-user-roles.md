# User Roles

## Platform Owner
- Access to all tenants
- Billing management
- Platform settings
- Not involved in day-to-day operations

## Tenant Admin (e.g., Showtime Pool Owner)
- Full access to their company's data
- Create and manage work orders
- Assign technicians
- View all reports
- Manage settings

## Office Staff
- Create and manage work orders
- Schedule visits
- View all properties
- Cannot access billing or platform settings

## Technician
- Mobile view only
- See only their assigned jobs
- Complete checklists, upload photos, write notes
- Flag estimate needed

## Read-Only Owner/Viewer
- Can view dashboard and reports
- Cannot create or modify records
- Useful for silent partners or investors
