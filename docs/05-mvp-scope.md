# MVP Scope — Phase 1–8

## In Scope for MVP

### Dashboard Shell (Phase 1)
- Navigation sidebar
- Dashboard layout
- Placeholder sections

### Work Orders (Phase 2)
- Create work order
- Assign technician
- Status lifecycle
- Work order detail view
- Link to property

### Property Profiles (Phase 3)
- Property record with address
- Pool equipment fields
- Access notes
- Link to GHL contact

### Technician Mobile View (Phase 4)
- Today's jobs list
- Job detail view
- Checklist completion
- Photo upload (placeholder)
- Notes
- Complete or flag estimate

### GHL Webhook Intake (Phase 5)
- Webhook endpoint
- Signature verification
- Work order creation from GHL data

### Status Sync Back to GHL (Phase 6)
- Job completed → update GHL
- Estimate needed → flag GHL opportunity

### Reporting (Phase 7)
- KPI cards
- Job status breakdown
- Technician summary

## Explicitly Out of Scope for MVP
- Route optimization
- Customer portal
- Invoicing and payments
- Multi-tenant billing
- Native mobile app
- Inventory
- AI assistant
- Dispatch board
- White-label theming
