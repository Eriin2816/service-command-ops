# What ServiceOps Handles

## ServiceOps Is Responsible For:
- Work orders (creation, assignment, lifecycle)
- Field jobs and visits
- Technician job routing and mobile view
- Job checklists (pool service templates)
- Before/after photo documentation
- Technician notes and completion reports
- Property profiles (address, equipment, access notes)
- Pool equipment records per property
- Recurring service visit tracking
- Estimate-needed handoff back to GHL
- Owner operations dashboard
- Job completion reports

## Data ServiceOps Owns:
- Work orders
- Visits
- Property profiles
- Equipment records
- Checklists
- Photos
- Technician notes
- Completion reports

## Data ServiceOps References From GHL (Read Only):
- GHL contact ID (linked to customer)
- GHL opportunity ID (linked to job)
- GHL calendar event ID (linked to appointment)
