# Testing Rules

- Manual test plan in qa/ before each phase
- QA checklist reviewed before marking phase complete
- GHL webhooks tested with real payloads (use test mode)
- All user roles tested for permission boundaries
- Technician view tested on actual phone (not just browser resize)
- Tenant isolation tested with two test tenants
- API error handling tested (GHL down, DB error)
- No phase ships without QA sign-off from launch checklist
