# Database Entities

| Entity | Purpose |
|--------|---------|
| tenants | Multi-tenant SaaS organizations |
| users | Platform users (all roles) |
| properties | Customer service locations (pools) |
| work_orders | Job records linked to properties and GHL contacts |
| visits | Individual service visit instances |
| checklist_items | Per-visit checklist completion records |
| technician_notes | Notes added by technicians on visits |
| photos | Before/after photo records per visit |
| estimate_handoffs | Records of estimate flags sent back to GHL |
| equipment_records | Pool equipment per property |
| recurring_schedules | Recurring visit schedule configurations |
