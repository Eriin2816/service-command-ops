# Glossary

| Term | Definition |
|------|-----------|
| Work Order | A job record for a specific property |
| Visit | A single service appointment within a work order |
| Property | A customer's service location (house with a pool) |
| Technician | A field worker who performs pool service |
| Checklist | A list of tasks to complete during a visit |
| Estimate Needed | A flag when a job requires a quote before proceeding |
| GHL | GoHighLevel — the CRM platform the client uses |
| Tenant | A company using the ServiceOps SaaS platform |
| Tenant Admin | The owner/manager of a tenant company |
| Source of Truth | The system that owns and controls a piece of data |
