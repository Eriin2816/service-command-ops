# MVP Requirements

## Must Have (Launch)
- [ ] Dashboard shell with navigation
- [ ] Work order list and detail views
- [ ] Work order creation and assignment
- [ ] Work order status lifecycle
- [ ] Property profiles with equipment records
- [ ] Technician mobile job list (today)
- [ ] Technician job detail with checklist
- [ ] Photo upload placeholder
- [ ] Technician notes
- [ ] Mark complete / flag estimate needed
- [ ] GHL webhook intake endpoint
- [ ] Estimate-needed handoff to GHL
- [ ] Owner dashboard with KPI cards

## Should Have (Launch If Possible)
- [ ] Recurring visit tracking
- [ ] Job completion report PDF
- [ ] Status sync back to GHL

## Nice to Have (Post-Launch)
- [ ] Advanced filtering and search
- [ ] Technician productivity report
- [ ] Route view on map
